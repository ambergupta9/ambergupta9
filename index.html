My Projects

  • Face Detection using OpenCV

    • This project uses raspberry pi 3 for processing and a camera for capturing the image.The image data set was reduced in dimentionality using principle component analysis(eigen faces approach).Recognition of faces was done by calculating the minimum euclidean distance between test feature vector and training feature vector.

  • Digital Spectrum Analyzer using IOT

    • This is an ambitious project which aims at developing a cheap alternative to benchtop spectrum analyzers.The main aim was make it wireless so that the data could be received on a server which can be used later to perform various kinds of analysis.It makes use of NodeMCU for sending the data and all the processing is done on the target device(Laptop).The graph is displayed on the target device hence reducing its cost.
